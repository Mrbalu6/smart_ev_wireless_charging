# Smart EV Wireless Charging & Battery Protection System

## Overview
An IoT-based smart EV charging system using wireless power transfer and battery protection technology.

The project monitors:
- Battery temperature
- Battery voltage
- Charging condition

using Arduino and ESP8266 WiFi module.

## Working Principle
The system uses resonant inductive coupling to transfer power wirelessly between transmitting and receiving coils.

The Arduino continuously monitors:
- Battery temperature
- Battery voltage

If temperature exceeds threshold:
- Charging stops automatically
- Relay disconnects battery
- Buzzer alert activates

All data is transmitted using ESP8266 WiFi module.
